# Ticket App - Vue

Features:

- Signup & Login
- User-scoped tickets
- CRUD with edit page (/tickets/:id/edit)
- Toast notifications via vue3-toastify

# Libraries

- react-hot-toast
- react-router
- tailwind-css

  Quick start:

1. npm install
2. npm run dev
